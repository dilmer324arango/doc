Instalación y configuración

La recomendación es usar yarn y no npm

    # Instalación global
    yarn global add vuepress@next
    npm install -g vuepress@next 
    
    # Instalación local 
    yarn add -D vuepress@next         
    npm install vuepress@next --save

Configuración

El archivo de configuración define las rutas de acceso a los archivos en : docs/.vuepress/config.js

    module.exports = {
        title: 'Documentación',
        description: "Documentación general",
        port : 2121,
        dest : "./publish",
        themeConfig:{
            nav: [
                { text: 'VuePress', link: '/vuePress/vuepress' }                
            ],
            sidebar: [
                {
                title: 'VuePress',
                collapsable: true,
                children: [
                    '/vuePress/vuepress'
                ]
                }
            ]
        },
        markdown:{
            lineNumbers: true,
            anchor: { permalink: false, permalinkBefore: false, permalinkSymbol: '' }
        }
    }
