Cargo

Cargo es el sistema de compilación y el administrador de paquetes de Rust


cargo --list                        # lista todos los comandos de cargo
cargo --version                     # Comprobar que cargo está instalado
cargo --help                        # Ayuda de cargo
cargo new nombre_proyecto           # Crear un nevo proyecto con cargo
cargo build                         # Compilar el proyecto
cargo install --list                # lista todos los paquetes instalados
cargo install --force <create>      # Sobreescribe el paquete si ya está instalado
cargo build --release               # Compila con optimizaciones
cargo update                        # Actualiza todas las dependencias
cargo uptdate -p rand               # Actualiza el paquete rand

# Utilitarios
cargo install --force cargo-watch   # Libreria para hacer watch del proyecto
cargo watch -x test                 # Ejecuta solo test
cargo watch -x run                  # Ejecuta el proyecto

# Test
cargo test --no-run                 # Compila pero no corre el test
cargo test foo_test                 # Ejecuta la prueba foo_test

Con el ejemplo anterior al compilar el proyecto se crea una carpeta target/debug, en esta carpeta queda el ejecutable con el nombre del proyecto(nombre_proyecto). Es necesario configurar el archiov launch.json para poder realizar un debugg.

{
    "version": "0.2.0",
    "configurations": [
        {
            "name": "(Windows) Launch",
            "type": "cppvsdbg",
            "request": "launch",
            "program": "${workspaceFolder}/target/debug/nombre_proyecto",
            "args": [],
            "stopAtEntry": false,
            "cwd": "${workspaceFolder}",
            "environment": [],
            "externalConsole": true
        }
    ]
}
