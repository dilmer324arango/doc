Cargo

Cargo es el sistema de compilación y el administrador de paquetes de Rust

# Comprobar que cargo está instalado
cargo --version
# Ayuda de cargo
cargo --help
# Crear un nevo proyecto con cargo
cargo new nombre_proyecto
cd nombre_proyecto

Con el ejemplo anterior al compilar el proyecto se crea una carpeta target/debug, en esta carpeta queda el ejecutable con el nombre del proyecto(nombre_proyecto). Es necesario configurar el archiov launch.json para poder realizar un debugg.

{
    "version": "0.2.0",
    "configurations": [
        {
            "name": "(Windows) Launch",
            "type": "cppvsdbg",
            "request": "launch",
            "program": "${workspaceFolder}/target/debug/nombre_proyecto",
            "args": [],
            "stopAtEntry": false,
            "cwd": "${workspaceFolder}",
            "environment": [],
            "externalConsole": true
        }
    ]
}
