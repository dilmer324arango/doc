Operaciones con cadenas

Imprimir en línea de comandos

string firstFriend   = "Maria";
string secondFriend  = "Sage";
string saludo        = "      Hello World!       ";
string saludoTrim    = greeting.Trim();
string sayHello      = saludoTrim.Replace("Hello", "Greetings");
string songLyrics    = "You say goodbye, and I say hello";
string A             = "a"
string B             = "z
// Impresión de una cadena en línea de comandos 
Console.WriteLine($"My friends are {firstFriend} and {secondFriend}");

// Length : Retorna la longigud de la cadena 
Console.WriteLine($"{saludoTrim.Length");  // 12

// Compare : Compara dos caddena si :
// A < B  : -1  
// B < A  :  1
// A = B  :  0
string A = "a"; string B = "z"; 
Console.WriteLine ($"{A.CompareTo(B)}");   //  -1
string A = "z"; string B = "a"; 
Console.WriteLine ($"{A.CompareTo(B)}");   //  1
string A = "x"; string B = "x"; 
Console.WriteLine ($"{A.CompareTo(B)}");   //  0
// true ignora si es mayusculas o minúscula 
string A = "Y"; string B = "y"; 
Console.WriteLine ($"{String.Compare(A,B, true)}");   //  0

// Concat
string A = "Hola "; string B  = "Mundo";
Console.WriteLine ($"{String.Concat(A,B)}");   //  "Hola Mundo"

// Equals : Permite saber si dos cadenas son iguales 
string A ="Hola"; string B ="Hola";
Console.WriteLine ($"'{ String.Equals(A,B)}'"); // 'True'

// IsNullOrEmpty
string A ="";
Console.WriteLine ($"'{ String.IsNullOrEmpty(A)}'"); // 'True'

// Trim 
Console.WriteLine($"{saludoTrim}");  // "Hello World!"

// Replace : busca una subcadena y la reemplaza 
Console.WriteLine($"{sayHello}");  // "Greetings World!"

// ToUpper
Console.WriteLine(sayHello.ToUpper()); // "GREETINGS WORLD!"

// ToLower
Console.WriteLine(sayHello.ToLower()); // "greetings world!"

// Contains : Busca la ocurrrencia de una cadena en otra retorna un booleano
Console.WriteLine (songLyrics.Contains ("goodbye"));   // True 
Console.WriteLine (songLyrics.Contains ("greetings")); // False

Listas

https://www.tutorialsteacher.com/csharp/csharp-list

Las listas (List) son colecciones de de tipo T (Generic), donde todos sus componentes son del mismo tipo.

Acciones que se pueden ejecutar en una lista

  • Add(T): void
  • AddRange(IEnumerable): void
  • BinarySearch(T): int
  • Clear(): void
  • Conatains(T): bool
  • CopyTo(T[], int): void
  • Exist(Predicate): bool
  • Find(Predicate): T
