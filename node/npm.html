Instalación de paquetes

Los paquetes se pueden instalar de manera local o global

# Instalacion  global
npm i -g  <package name>

# Instalación local : en dependencies
npm i <package name>

# Instalación local : en dev dependencies
npm i <package name> --save-dev

# Instalar una version específica
npm install <package name>@1.8.2

Desinstalar un paquete

Remover un paquete del proyecto actual

# Desinstalar un paquete local
npm uninstall  <package name>
npm un <package>
npm list
# (empty)

# Desinstalar un paquete global
npm uninstall  -g <package name>
npm list -g --depth=0

Ejecutar un comando desde npm

En el archivo package.json colocar una sección script así :

"scripts": {
    "d": "vuepress dev docs",
    "b": "vuepress build docs",
    "p": "live-server --port=7654 -q"
},

# En línea de comandos ejecutar :
npm run <opción>  : npm run p

# Ejecutar una instrucción de un paquete en node_modules

npx tsc

Npx

Npx es un ejecutador de paquetes binarios de npm

# Ejemplo si tenemos el paquete ESLint instalado como :
./node_modules/.bin/eslint yourfile.js

# La ejecución se puede hacer :
npx eslint yourfile.js

# Lista de paquetes que se pueden ejecutar
https://github.com/junosuarez/awesome-npx
