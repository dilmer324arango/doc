Keyboard shortcuts

Funciones rápidas del teclado


    Navegar por el explorador de archivos           Crtl + 0 (cero del teclado)
    Abrir y cerrar terminal                         Ctrl + ñ
    Abrir y cerrar terminal                         Ctrl + j 
    Abrir cerrar el sidebar                         Ctrl + b
    Navegar entre archivos                          Ctrl + p 
    Navegar entre funciones de archivo              Ctrl + Shift + o (vocal o)
    Abrir una nueva ventana                         Ctrl + 2 
    Cerrar la ventana actual                        Ctrl + F4
    Ventana de propiedades de carpeta/archivo       Ctrl + F10
    Reveal in explorer(seleccionar carpeta)         Ctrl + r 
    Cerrar todas las ventanas                       Ctrl + k + w
    Explorador de archivos                          Ctrl + Shift + e
    Navegar entre ventanas                          Ctrl + Alt -> | <-
