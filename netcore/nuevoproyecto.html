Construcción de un proyecto WebApi con tres capas

/*
    1.  Crea la solución -n es para indicar el nombre
    2.  Crea el proyecto Webapi -o para crear una nueva carpeta con nombre app.WebApi
        -au None : quitar la autenticación
    3.  Crea una libreria de clases para la capa de Entidades de dominio
    4.  Crea una libreria de clases para la capa de datos
    5.  Agrega a la solución el proyecto WebApi
    6.  Agrega a la solución el proyecto de Entidades
    7.  Agrega a la solución el proyecto de Datos
    8.  Agrega las referencias de Entidades y Datos al proyecto WebApi
    9.  Agrega la referencia de Entidades al proyecto de Datos
    10. Agrega la referencias de las librerias SqlServer y EntityFrameworkCore.Tools a la capa de Datos
    11. En la capa de Entidades agregar la libreria System.ComponentModel.Annotations

*/

/* 1. */    dotnet new sln -n appSolution
/* 2. */    dotnet new webapi -au None -o app.WebApi --no-https
/* 3. */    dotnet new classlib -o app.Entidades
/* 4. */    dotnet new classlib -o app.Datos
/* 5. */    dotnet sln appSolution.sln add ./app.WebApi/app.WebApi.csproj
/* 6. */    dotnet sln appSolution.sln add ./app.Entidades/app.Entidades.csproj
/* 7. */    dotnet sln appSolution.sln add ./app.Datos/app.Datos.csproj
/* 8. */    dotnet add app.WebApi/app.WebApi.csproj reference app.Entidades/app.Entidades.csproj
            dotnet add app.WebApi/app.WebApi.csproj reference app.Datos/app.Datos.csproj
/* 9. */    dotnet add app.Datos/app.Datos.csproj reference app.Entidades/app.Entidades.csproj
/* 10.*/    dotnet add package Microsoft.EntityFrameworkCore.SqlServer
            dotnet add package Microsoft.EntityFrameworkCore.Tools
/* 11.*/    dotnet add package System.ComponentModel.Annotations

Runtime IDentifier Catalogo

RID Runtime IDentifier es el identificador que se usa para inicarle al compilador en que plataforma se quiere el ensamblado final https://docs.microsoft.com/en-us/dotnet/core/rid-catalog

Windows
  Portable (.NET Core 2.0 or later versions)
      win-x64
      win-x86
      win-arm
      win-arm64

  Windows 7 / Windows Server 2008 R2
      win7-x64
      win7-x86

  Windows 8.1 / Windows Server 2012 R2
      win81-x64
      win81-x86
      win81-arm

  Windows 10 / Windows Server 2016
      win10-x64
      win10-x86
      win10-arm
      win10-arm64

Linux
  Portable (.NET Core 2.0 or later versions)
      linux-x64    (Most desktop distributions like CentOS, Debian, Fedora, Ubuntu and derivatives)
      linux-musl-x64 (Lightweight distributions using musl like Alpine Linux)
      linux-arm (Linux distributions running on ARM like Raspberry Pi)

  Red Hat Enterprise Linux
      rhel-x64 (Superseded by linux-x64 for RHEL above version 6)
      rhel.6-x64 (.NET Core 2.0 or later versions)

  Tizen (.NET Core 2.0 or later versions)
      tizen
      tizen.4.0.0
      tizen.5.0.0

macOS
  Portable (.NET Core 2.0 or later versions)
    osx-x64 (Minimum OS version is macOS 10.12 Sierra)

  macOS 10.10 Yosemite
    osx.10.10-x64

  macOS 10.11 El Capitan
    osx.10.11-x64

  macOS 10.12 Sierra (.NET Core 1.1 or later versions)
    osx.10.12-x64

  macOS 10.13 High Sierra (.NET Core 1.1 or later versions)
    osx.10.13-x64

  macOS 10.14 Mojave (.NET Core 1.1 or later versions)
    osx.10.14-x64

Debugger

Para realizar el debugger es necesario cambiar el archivo .vscode/launch.json

{
  "version": "0.2.0",
  "configurations": [
    {
      "name": ".NET Core Launch (web)",
      "type": "coreclr",
      "request": "launch",
      "preLaunchTask": "build",
      "program": "${workspaceFolder}/app.WebApi/bin/Debug/netcoreapp2.2/win7-x64/app.WebApi.exe",
      "args": [],
      "cwd": "${workspaceRoot}",
      "stopAtEntry": false,
      "internalConsoleOptions": "openOnSessionStart",
      "launchBrowser": {
        "enabled": false,
        "args": "${auto-detect-url}",
        "windows": {
          "command": "cmd.exe",
          "args": "/C start ${auto-detect-url}"
        },
        "osx": {
          "command": "open"
        },
        "linux": {
          "command": "xdg-open"
        }
      },
      "env": {
        "ASPNETCORE_ENVIRONMENT": "Development"
      },
      "sourceFileMap": {
        "/Views": "${workspaceRoot}/Views"
      }
    },
    {
      "name": ".NET Core Attach",
      "type": "coreclr",
      "request": "attach",
      "processId": "${command:pickProcess}"
    }
  ]
}

Tareas

Las tareas se ejecutan desde el archivo .vscode/tasks.json

{
    "version": "2.0.0",
    "tasks": [
        {
            "label": "build",
            "command": "dotnet",
            "type": "process",
            "group": {
                "kind": "build",
                "isDefault": true
            },
            "args": [
                "build",
                "${workspaceFolder}/Project/Project.csproj"
            ],
            "problemMatcher": "$msCompile"
        },
        {
            "label": "build tests",
            "command": "dotnet",
            "type": "process",
            "group": "build",
            "args": [
                "build",
                "${workspaceFolder}/Project.Tests/Project.Tests.csproj"
            ],
            "problemMatcher": "$msCompile"
        },
        {
            "label": "test",
            "command": "dotnet",
            "type": "process",
            "group": {
                "kind": "test",
                "isDefault": true
            },
            "args": [
                "test",
                "${workspaceFolder}/Project.Tests/Project.Tests.csproj",
                "/p:CollectCoverage=true",
                "/p:CoverletOutputFormat=\"opencover,lcov\"",
                "/p:CoverletOutput=../lcov"
            ],
            "problemMatcher": "$msCompile"
        },
        {
            "label": "publish win",
            "command": "dotnet",
            "group": "none",
            "args": [
                "publish",
                "${workspaceRoot}/Project/Project.csproj",
                "-o",
                "${workspaceRoot}/Library/win/",
                "-c",
                "release",
                "-r",
                "win10-x64"
            ],
            "problemMatcher": "$msCompile"
        },
        {
            "label": "publish mac",
            "command": "dotnet",
            "args": [
                "publish",
                "${workspaceRoot}/Project/Project.csproj",

                "-r",

            "label": "pack",

                "${workspaceRoot}/Project/Project.csproj",

                "release"

            "problemMatcher": "$msCompile"
