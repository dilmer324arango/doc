Recomendación

Yarn es un gestor de paquetes, más rápido y seguro que npm. Primero es necesario instalarlo para ello leer yarn

Inicialización en un proyecto

    # con esto se crea el package.json
    yarn init
    # es recomendable crer el package.json con npm
    npm i -y

Agregar a dependencias

    # devDependencies
    yarn add [package] --dev        ó    yarn add [package] -D 
    # peerDependencies
    yarn add [package] --peer       ó    yarn add [package] -P
    # optionalDependencies 
    yarn add [package] --optional   ó    yarn add [package] -O

Instalación de paquetes

    # global
    yarn global add [package]
    # local
    yarn            ó   yarn install
    #forzar a una nueva descarga de todos los paquetes 
    yarn install --force
    # instalar solo las dependencias de producción
    # Yarn no instalará ningún paquete listado en devDependencies
    yarn install --production   ó  yarn install --prod
